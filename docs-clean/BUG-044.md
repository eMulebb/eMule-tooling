## Summary

Download source rows could remain visible after their backing source object was
removed from the current download state.

## Mainline Outcome

Landed on `main` in commit `5b1fd1b` (`BUG-044 guard download-source stale
rows`).

Download source list paths now verify current ownership before dereferencing
row payloads.
