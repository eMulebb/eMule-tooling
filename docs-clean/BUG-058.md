## Summary

Localized tree option labels could insert the exact separator text used by the
value parser, breaking value extraction in some locales.

## Mainline Outcome

Landed on `main` in commit `4850328` (`BUG-058 protect tree option value
separators`).

Tree option edit labels now avoid embedding the parser separator in user-visible
label text.
