## Summary

Closing all search result tabs could leave Kad keyword searches running in the
background and block the same keyword from being searched again.

## Mainline Outcome

Landed on `main` in commit `f53e471` (`BUG-057 stop orphan Kad keyword
searches`).

`CSearchManager` now exposes all-keyword-search cleanup and the Search Results
window calls it when deleting all searches.
