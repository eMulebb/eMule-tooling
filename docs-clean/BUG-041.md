## Summary

The Known Clients list could hold row payload pointers after the corresponding
client object was no longer live.

## Mainline Outcome

Landed on `main` in commit `482aa8f` (`BUG-041 guard known-clients stale
rows`).

The list now validates item data before using it, returning a safe empty row
when the cached pointer is stale.
