## Summary

When an obfuscated server connection timed out, the client could fail to retry
the same server promptly with a plain connection.

## Mainline Outcome

Landed on `main` in commit `9cb92a7` (`BUG-062 retry plain server connect after
obfuscated timeout`).

The server-connect path now retries the current candidate without obfuscation
when the obfuscated attempt times out.
