## Summary

Source Exchange v2 superseded v1 years ago. The codebase still sends and handles
v1 for backward compatibility with very old clients, with heavy conditional
branching throughout `BaseClient.cpp` and `DownloadClient.cpp`. Once the minimum
supported client version is set to v2-capable, all v1 paths become dead weight.

## Estimated Removable Code

~200ŌĆō300 lines across `BaseClient.cpp` and `DownloadClient.cpp`.

## Pre-conditions

Before removing v1 branches:
1. Confirm minimum supported client version is v2 Source Exchange capable.
2. Verify no current peers on the network send v1-only source exchange (unlikely
   given v2 has been standard for 10+ years).

## Implementation Steps

1. Remove `m_bySourceExchange1Ver` member and all assignments.
2. Remove `uSourceExchange1Ver` constant and all uses.
3. Remove `if (SupportsSourceExchange2())` branching ŌĆö keep only the v2 path.
4. Delete the dead `wasSkippedDueToSourceExchange` tracking code.
5. Remove the two dead commented-out debug log calls in `DownloadClient.cpp:481, 557`.
6. Clean up remaining SX v1 debug log strings.
